    HTML

    Significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem padrão para criar páginas e aplicações web. É uma linguagem de marcação, o que significa que ela usa tags para definir elementos dentro de um documento. Essas tags são usadas para formatar e organizar o conteúdo da página.

    CSS

    Significa Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo usada para descrever a aparência e formatação de um documento escrito em HTML. CSS é usada para controlar o layout de várias páginas web de uma só vez, permitindo que você altere a aparência de todo o site mudando apenas um arquivo.

    JavaScript

    É uma linguagem de programação que permite implementar funcionalidades mais complexas em páginas web. Enquanto HTML e CSS são linguagens que dão estrutura e estilo à página web, respectivamente, JavaScript dá à página web elementos interativos que os usuários podem se engajar. Esses elementos interativos podem incluir slideshows de imagens, formulários de envio com mensagens de validação, conteúdo que muda e atualiza em tempo real, efeitos animados e gráficos interativos.

    PHP

    Significa: Hypertext Preprocessor (Pré-processador de Hipertexto PHP), é uma linguagem de programação de código aberto amplamente usada para desenvolvimento web. É executada no servidor, o que significa que a execução (leitura) do código PHP é feita no servidor onde a página é hospedada. PHP pode gerar conteúdo dinâmico na página, como texto, HTML, CSS e JavaScript, e pode interagir com vários sistemas de banco de dados.

    Node.JS

    É uma plataforma construída sobre o motor JavaScript do Chrome para facilmente construir aplicações de rede rápidas e escaláveis. Node.js usa um modelo de I/O direcionada a evento não bloqueante, que o torna leve e eficiente, ideal para aplicações em tempo real com troca intensa de dados através de dispositivos distribuídos. Ele permite que o JavaScript seja executado no lado do servidor, tendo assim acesso a coisas como sistema de arquivos, servidores HTTP e bancos de dados, coisas que o navegador normalmente não permite por questões de segurança.

    MySQL

    É um sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto baseado em SQL (Structured Query Language). SQL é a linguagem mais popular para adicionar, acessar e gerenciar conteúdo em um banco de dados. É conhecido por sua rapidez e confiabilidade. Ele é comumente usado em aplicações web para armazenar dados e é uma componente das pilhas de desenvolvimento web amplamente utilizadas, como LAMP (Linux, Apache, MySQL, PHP/Python/Perl) e MERN (MongoDB, Express.js, React.js, Node.js).
